Сайт Форум для обсуждения музыки

Сайт преднозначен для малоизвестных исполнителей, которые могут делиться своими работами. Так же сайт предназначен для тех пользователей,
которые ищут новую музыку.
На сайте можно выкладывать посты, музыку, устраивать мероприятия

Для пользования сайта необходима регистрация
Для сохранения изображений в песни и аватарах профиля необходимо нажимать на кнопку <<Обновить>>, иначе изображение не сохранится
Для изменения места на карте необходимо нажимать на кнопку <<Обновить>>, чтобы сохранять изображение карты указанного места
